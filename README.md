# C++ Interview Preparation Workspace

This workspace is organized to help you practice and review key C++ topics for technical interviews.

## Folders & Topics

- **cuda/**: GPU programming with CUDA
- **network/**: Network programming (TCP/UDP, sockets)
- **poxis/**: POSIX I/O and system programming

### Recommended Additional Folders
- **algorithms/**: Sorting, searching, graph algorithms, etc.
- **concurrency/**: Multithreading, synchronization, thread safety
- **memory/**: Smart pointers, manual memory management, RAII
- **oop/**: Object-oriented programming, design patterns
- **templates/**: Generic programming, template metaprogramming
- **stl/**: Standard Template Library usage
- **build/**: Build systems (CMake, Makefiles)
- **tests/**: Unit testing frameworks

## Interview Focus Areas
- Core C++ syntax and semantics
- Algorithms and data structures
- Multithreading and concurrency
- Memory management and smart pointers
- OOP principles and design patterns
- Networking and system programming
- Code quality, error handling, and testing

## How to Use
- Add code examples and exercises in each folder.
- Use README files in subfolders for guidance and resources.
- Practice writing, debugging, and explaining code.
